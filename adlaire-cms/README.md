# Adlaire Static CMS

**現在のバージョン: Ver.3.0-47**

軽量・高セキュリティなフラットファイル型静的 CMS。

PHP 8.3+ / TypeScript (Deno) / JSON フラットファイルストレージ。外部データベース不要。共有サーバーで動作。

## 主要機能

- ブロックエディタ + Markdown 対応
- 静的サイト生成（差分ビルド）
- マスター管理者モデル（最大3名）
- REST API
- 多言語化（日本語・英語）
- API キー認証システム
