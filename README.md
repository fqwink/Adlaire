# Adlaire Group 統合リポジトリ

Adlaire Group 及び 倉田和宏が関わる全てのプロジェクト開発の統合リポジトリ。

## プロジェクト一覧

| ディレクトリ | プロジェクト | 概要 |
|-------------|------------|------|
| `Adlaire Static CMS/` | **Adlaire Static CMS** | 軽量フラットファイル型静的 CMS |
| `Adlaire License Server/` | **Adlaire License Server** | API キー認証・認可管理システム |
| `Adlaire BaaS/` | **Adlaire BaaS** | 標準 BaaS 連携基盤 |
| `Adlaire Deploy/` | **Adlaire Deploy** | セルフホスト型デプロイプラットフォーム |
| `Adlaire Portal System/` | **Adlaire Portal System** | 軽量サーバーレス社内ポータルシステム |
