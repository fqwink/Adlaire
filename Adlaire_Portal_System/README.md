# Adlaire ポータルシステム

## 📌 概要
**Adlaire Portal System**は、サーバー不要で動作する軽量な社内ポータルシステムです。
HTMLとJavaScriptのみで構築されており、ブラウザだけで完結します。

## ✨ 特徴
- 🚀 サーバーレス: HTMLファイルを開くだけで動作
- 💾 **自動保存: LocalStorageで設定を自動保存 (NEW!)**
- 📤 **インポート/エクスポート: JSON形式でバックアップ可能 (NEW!)**
- 🎨 カスタマイズ可能: テーマカラー、リンク、カテゴリを自由に編集
- 📱 レスポンシブ対応: PC・スマートフォン両対応
- 🔍 検索機能: リアルタイムでリンクを検索
- 🎯 シンプル設計: 複雑な設定不要

## 📂 ファイル構成
```
adlaire_portal/
├── portal.html    # メイン表示画面
├── edit.html      # 設定エディタ
├── data.js        # 設定データ
└── README.md      # このファイル
```

## 🚀 使い方

### 1. セットアップ
1. このフォルダを任意の場所に配置してください
2. `portal.html` をブラウザで開くとポータルが表示されます

### 2. 設定の編集

#### 方法1: LocalStorage自動保存 (推奨) 🆕
1. `edit.html` をブラウザで開きます
2. 左側のエディタでリンクやカテゴリを編集します
3. 右側のプレビューで即座に確認できます
4. 「💾 保存 (自動適用)」ボタンをクリック
5. `portal.html` を再読み込みすると変更が即座に反映されます

#### 方法2: data.js手動更新 (従来方式)
1. `edit.html` をブラウザで開きます
2. 左側のエディタでリンクやカテゴリを編集します
3. 右側のプレビューで即座に確認できます
4. 「💾 data.js形式で保存 (旧形式)」ボタンをクリック
5. ダウンロードされた `data.js` を既存のファイルと置き換えます
6. `portal.html` を再読み込みすると変更が反映されます

### 3. バックアップとリストア 🆕
- **エクスポート**: 「📤 エクスポート (JSON)」で設定をバックアップ
- **インポート**: 「📥 インポート」で以前の設定を復元
- **デフォルトに戻す**: 「🔄 デフォルトに戻す」で初期設定に戻す

### 4. 共有サーバーへの配置
- フォルダごと共有サーバー（共有フォルダ）にコピーすれば、
  チーム全員で同じポータルにアクセスできます

## ⚙️ カスタマイズ

### テーマカラーの変更
編集画面の「テーマカラー」から好きな色を選択できます。
デフォルトは **#00a968**（グリーン）です。

### カテゴリの追加
編集画面下部の「＋ 新しいカテゴリを追加」ボタンから追加できます。

### リンクの追加
各カテゴリ内の「＋ リンクを追加」ボタンから追加できます。

### アイコンの変更
リンクのアイコン部分をクリックすると、30種類の絵文字から選択できます。

## 🔒 セキュリティについて
- 本システムは基本的なXSS対策を実装していますが、
  信頼できるユーザーのみが編集できる環境で使用してください
- URLは `http://` または `https://` で始まるもののみ有効です

## 🆘 トラブルシューティング

### Q. 保存したのに変更が反映されない
A. LocalStorageに保存後、ブラウザのキャッシュをクリアしてから再読み込みしてください。
   それでも反映されない場合は、ブラウザのコンソールでエラーが出ていないかチェックしてください。

### Q. LocalStorageとdata.jsのどちらが優先される?
A. LocalStorageに保存されたデータが優先されます。
   デフォルト設定に戻したい場合は「🔄 デフォルトに戻す」ボタンを使用してください。

### Q. 編集画面でプレビューが表示されない
A. `data.js` が同じフォルダに存在するか確認してください。
   ブラウザのコンソールでエラーが出ていないかチェックしてください。

### Q. インポートしたファイルが読み込めない
A. 正しいJSON形式のファイルか確認してください。
   「📤 エクスポート」ボタンで出力したファイルのみインポート可能です。

### Q. スマホで表示が崩れる
A. ブラウザの表示倍率を100%に設定してください。
   古いブラウザの場合は最新版に更新してください。
