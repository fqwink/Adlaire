# Adlaire Portal System

**Ver.1.1**

サーバー不要で動作する軽量な社内ポータルシステム。
HTML5 + JavaScript（バニラ）のみで構築されており、ブラウザだけで完結する。

---

## 特徴

- **サーバーレス**: HTMLファイルをブラウザで開くだけで動作する。
- **LocalStorage 自動保存**: 設定変更をブラウザのローカルストレージに自動保存する。
- **インポート/エクスポート**: JSON形式で設定のバックアップ・復元が可能。
- **カスタマイズ**: テーマカラー・リンク・カテゴリを自由に編集できる。
- **レスポンシブ**: PC・スマートフォン両対応。
- **リアルタイム検索**: 入力に応じてリンクをフィルタリングする。
- **外部依存なし**: CDN・外部ライブラリを一切使用しない。

## 使い方

### 起動

`portal.html` をブラウザで開く。

### 設定の編集

`edit.html` をブラウザで開き、左パネルで編集後「保存」ボタンをクリックする。
`portal.html` を再読み込みすると変更が反映される。

### バックアップとリストア

- **エクスポート**: 「エクスポート (JSON)」ボタンで設定をバックアップする。
- **インポート**: 「インポート」ボタンで以前の設定を復元する。
- **デフォルトに戻す**: 「デフォルトに戻す」ボタンで `data.js` の初期設定に戻す。

### 共有サーバーへの配置

フォルダごと共有フォルダにコピーすることで、チームで共有できる。
