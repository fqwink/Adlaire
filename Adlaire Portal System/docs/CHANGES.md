# Adlaire Portal System — CHANGES

---

## Ver.1.1（2026-02-27）

### 追加機能

- LocalStorage 自動保存機能を追加
- データインポート/エクスポート機能（JSON形式）を追加
- デフォルト設定に戻す機能を追加

### 機能改良

- 設定管理の大幅な改善（LocalStorage を優先、`data.js` はデフォルト値として使用）

---

## Ver.1.0（2026-02-14）

### 追加機能（初版リリース）

- メイン表示画面（`portal.html`）: カテゴリ別リンク・ニュース・リアルタイム検索・サイドバーナビ
- 設定エディタ（`edit.html`）: 分割レイアウト・プレビュー・リンク/カテゴリ/テーマ編集
- 設定データファイル（`data.js`）: `PORTAL_CONFIG` 定数によるデフォルト設定
- レスポンシブ対応（PC・スマートフォン）
- ダークモード自動切り替え（`prefers-color-scheme`）
